# Timeless-decor---landing-page
:-Timeless Decor — Landing Page
A fully responsive, aesthetic landing page designed for a curated home decor affiliate store. Built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies.
✨ Overview
Timeless Decor is a concept affiliate storefront that curates minimal, modern home decor products sourced from Amazon. This landing page was designed to feel warm, elegant, and trustworthy — converting visitors into buyers through clean UI and intentional design choices.
:- Design Highlights

Color Palette — White and warm beige tones for a cozy, premium feel
Typography — Cormorant Garamond for display headings, Jost for body text
Hero Section — Bold headline with an animated card carousel that auto-shuffles right-to-left every 3.5 seconds with a smooth cubic-bezier ease
Stats Bar — Trust signals highlighting curated picks, monthly updates, budget-friendly finds, and Amazon-trusted products
Product Grid — 3 curated decor cards with "View on Amazon" CTA buttons, no pricing shown
Scroll Animations — Elements fade and slide up on scroll using Intersection Observer

:- Built With

HTML5
CSS3 (custom properties, keyframe animations, clip-path)
Vanilla JavaScript (Intersection Observer, carousel logic)
Google Fonts
